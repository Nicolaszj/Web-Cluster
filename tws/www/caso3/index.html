Caso de Prueba 3

Esta pagina enlaza a un archivo de aproximadamente 1 MB.

Este caso verifica que el servidor puede transferir archivos grandes correctamente usando el bucle de lectura por chunks (bloques de 8KB).

Sin el bucle de envio, la transferencia se truncaria silenciosamente cuando el buffer del socket del kernel se llenara.

Descarga de prueba

Descargar archivo_1mb.bin (~1 MB)

Genera este archivo en Linux con: dd if=/dev/urandom of=archivo_1mb.bin bs=1024 count=1024
