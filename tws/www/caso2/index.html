Esta pagina carga multiples imagenes. Cada imagen genera una peticion GET separada al servidor. Esto permite verificar que el servidor maneja correctamente peticiones concurrentes desde el mismo browser.

Sobre este caso

Un browser moderno puede abrir 6 conexiones paralelas al mismo origen. Si abres DevTools → Network, veras las peticiones de las imagenes ejecutandose en paralelo. Cada una de esas conexiones es atendida por un hilo diferente del TWS.
